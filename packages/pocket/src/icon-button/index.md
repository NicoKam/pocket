## API

| 属性     | 说明         | 类型                                             | 默认值      |
| -------- | ------------ | ------------------------------------------------ | ----------- |
| checked  | 是否选中状态 | `boolean`                                        | `false`     |
| size     | 按钮大小     | `'x-small'` / `'small'` / `'middle'` / `'large'` | `'middle'`  |
| disabled | 禁用按钮     | `boolean`                                        | `false`     |
| theme    | 主题色       | `'default'` / `'dark'`                           | `'default'` |
